# OR-LIFE: AI Rural Home Care Coordination System

**AI that acts before emergencies become tragedies.**

---

## Why OR-LIFE?

Healthcare is shifting.

- Aging populations are increasing
- Healthcare workforce is shrinking
- Emergencies happen at home — not in hospitals

So the question becomes:

**What if care could start before the ambulance arrives?**

---

## What is OR-LIFE?

OR-LIFE is a **multi-agent AI system** designed for rural and home-based care.

It transforms isolated homes into **connected points of care**.

---

## Core Capabilities

### 🚨 Emergency Response
- Detects high-risk deterioration
- Activates EMS (112)
- Alerts neighbors and community
- Generates structured emergency intelligence

---

### 🏡 Home Monitoring
- Voice-based patient check-ins
- Detects early deterioration signals
- Escalates before collapse

---

### 📊 Longitudinal Intelligence
- 1 year → 6 months → 3 months → current
- Detects patterns, not isolated values
- Predicts deterioration trajectory

---

## 🧠 Multi-Agent Architecture

OR-LIFE operates using coordinated AI agents:

- Doctor Agent → clinical context
- Risk Agent → risk scoring
- Monitoring Agent → trend analysis
- Escalation Agent → decision routing
- EMS Agent → emergency coordination

All decisions are handled by:

**A2A + MCP Decision Engine**
